pub unsafe fn st_foreach_check(
    table: *mut st_table,

    arg: st_data_t, 
    _never: st_data_t
) -> c_int
Expand description

Traverse all entries in table table calling func with current entry key and value and zero. If the call returns ST_STOP, stop traversing. If the call returns ST_DELETE, delete the current entry from the table. In case of ST_CHECK or ST_CONTINUE, continue traversing. The function returns zero unless an error is found.

The behavior is a bit different from st_foreach when ST_CHECK is returned from func and when the current element is removed during traversing.

Header declaration

int st_foreach_check(st_table *, int (*)(ANYARGS), st_data_t, st_data_t);

Safety

table must be non-null and point to a valid st_table created by the st_init_table family of functions.

func must be a callback function with the st_foreach_callback_func signature.
